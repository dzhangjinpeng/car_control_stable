# 稳定版架构

## 目标

对齐 Python 版的关键功能，同时把实时控制主链路放进 C++。

```text
输入源 -> 输入平滑 -> 控制状态 -> 模式控制 -> 安全限幅 -> 电机客户端 -> 遥测
```

## 控制核心

C++ 核心负责：

- 固定周期循环
- 输入平滑和摇杆曲线
- 驱动速度斜坡
- mode0 / mode1 / mode2
- 转向锁
- 前进/倒车方向锁
- 角色到电机 ID 的映射
- 达妙硬件下发
- JSONL 遥测

## Python 工具

Python 保留在非实时链路：

- 遥测 API
- 远程输入发送器
- 校准脚本入口
- 配置和健康检查

## 前端

前端只读展示：

- 模式、输入源、链路状态
- 转向锁、方向锁、急停、限幅
- 远程序号、延迟、过期状态
- 电机目标、实测、误差
- 配置、校准报告、历史遥测

## 仍需真机确认

- USB-CANFD 链路
- 达妙电机 ID
- 轮位映射
- 反向驱动列表
- 手柄实际轴号和按键号
